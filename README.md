## 🚀 快速本地体验

在项目目录下启动本地静态服务（任选其一）：

```bash
cd milk-tea-mystic-selector
python3 -m http.server 8080
# 或者
npx serve . -p 8080
```

然后访问 `http://localhost:8080`。

> 提示：项目是纯前端零依赖，直接用浏览器打开 `index.html` 也可以。
